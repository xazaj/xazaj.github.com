第一章 Rails3.2 环境配置

1、用rvm安装ruby&rails

Ruby安装：

    $ rvm get head && rvm reload
    $ rvm install 1.9.3

创建rails gemsets

    $ rvm use 1.9.3@rails3tutorial2ndEd --create --default

更新RubyGems版本为本教程使用的版本：

    $ gem update --system 1.8.24

用subl编辑gemrc

    $ subl ~/.gemrc

编辑gemrc文件：

install： –no-rdoc –no-ri

update: –no-rdoc –no-ri
